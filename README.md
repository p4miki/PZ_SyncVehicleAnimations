# Vehicle Animation MP Fix

Fix to synchronize vehicle animation in multiplayer for Project Zomboid.

## Mod info

- **Name:** Vehicle Animation MP Fix
- **ID:** `SyncVehicleAnim`
- **Description:** Fix to synchronize vehicle animation in MP.

## Install

- **Steam Workshop:** Subscribe as normal.
- **Manual:** Place the mod folder so Project Zomboid can see the `mod.info`.

## Usage

Enable the mod in:

- **Main Menu → Mods** (singleplayer), and/or
- **Server Settings → Mods** (multiplayer)

No additional configuration is required.

## Development notes

Lua entry points live under:

- `42/media/lua/client/`
- `42/media/lua/server/`
